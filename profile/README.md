### 📁 Estructura de Proyectos Estándar
En JEGA-BYTE mantenemos el orden para garantizar la escalabilidad:
* `01_HW`: Archivos de diseño electrónico (EDA).
* `02_FW`: Código fuente y librerías.
* `03_DOCS`: Datasheets, cálculos y manuales.
* `04_OUTPUTS`: Gerbers, BOM y Pick&Place para fabricación.
