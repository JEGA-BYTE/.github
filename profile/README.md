# ⚡ JEGA-BYTE | Electronic Prototyping & PCB Design

Bienvenido al centro de ingeniería de **JEGA-BYTE**. Nos especializamos en convertir conceptos en circuitos reales a través de un diseño de PCB profesional y prototipado rápido.

---

### 🔬 Nuestras Capacidades
* **Diseño de Hardware:** Desarrollo de esquemáticos y ruteado de PCBs multicapa.
* **Prototipado Rápido:** Montaje de placas y pruebas de concepto (PoC).
* **Sistemas Embebidos:** Programación de firmware para microcontroladores (ESP32, STM32, Arduino).
* **DFM (Design for Manufacturing):** Optimización de diseños para producción industrial.

### 🛠️ Herramientas de Ingeniería
* **EDA:** KiCad / Altium Designer / Autodesk Eagle
* **Hardware:** C / C++ / MicroPython / CircuitPython
* **Simulación:** Proteus / SPICE
