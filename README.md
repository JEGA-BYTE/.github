# <img src="https://raw.githubusercontent.com/JEGA-BYTE/.github/main/profile/logo.png" width="50" /> JEGA-BYTE | Hardware Engineering Hub

### 🛠️ "Ingeniería con precisión, innovación en cada trazo"
Bienvenido al centro de operaciones de **JEGA-BYTE**. Somos una organización enfocada en el ciclo completo de desarrollo electrónico: desde la concepción del esquemático hasta el firmware de bajo consumo y el prototipado rápido de PCBs.

---

### 🚀 Capacidades Técnicas
| Área de Especialidad | Tecnologías & Herramientas |
| :--- | :--- |
| **Diseño de Hardware** | KiCad, Altium, PCBs Multicapa (L4/L6), DFM |
| **Sistemas Embebidos** | STM32 (HAL/LL), ESP32, RTOS, ARM Cortex-M |
| **Firmware & Software** | C/C++, MicroPython, Rust, IoT Protocols (MQTT/LoRa) |
| **Prototipado Industrial** | Soldadura SMD de precisión, Análisis de Señal, EMI/EMC |

---

### 📋 Estándar de Estructura JEGA
Para mantener la escalabilidad en nuestros proyectos, todos los repositorios siguen esta jerarquía:
* 📂 **`01_HW`**: Archivos de diseño electrónico (Esquemáticos, Layout, Librerías).
* 📂 **`02_FW`**: Código fuente, scripts de compilación y binarios.
* 📂 **`03_DOCS`**: Datasheets de componentes críticos y manuales de usuario.
* 📂 **`04_OUTPUTS`**: Archivos Gerber, BOM y Pick&Place listos para fábrica.
